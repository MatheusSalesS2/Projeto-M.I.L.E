 - Projeto M.I.L.E - Circuitos Digitais

Este projeto foi desenvolvido utilizando o software **wiRedPanda** para a disciplina de Circuitos Digitais.

 - Sobre o Projeto
Sistema inteligente e de baixo custo para triagem automatizada de plásticos (LDPE, HDPE e PS) utilizando sensores multiespectrais (NIR), lógica digital e automação pneumática alinhado ao ODS 12 da ONU.

 - Estrutura do Repositório
* Circuito do Projeto: Contém os arquivo de simulação no wiRedPanda do projeto final junto dos outros mini circuitos que auxiliaram na composição do projeto final.
* Documento do Projeto: Contém a documentação detalhada em PDF.

 - Como rodar o projeto
1. Baixe e instale o [wiRedPanda](https://github.com/GIBIS-UNIFESP/wiRedPanda).
2. Abra o software e importe o arquivo do circuito que está na pasta Circuito M.I.L.E, o circuito com o programa principal é o CIRCUITOOK.panda.
3. Extraia os arquivos *na  mesma pasta* e abra o CIRCUITOOK.panda.
